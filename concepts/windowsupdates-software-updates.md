# <a name="software-updates-with-the-windows-update-for-business-deployment-service"></a>Обновления программного обеспечения с помощью Windows службы развертывания для бизнеса

Обновления программного обеспечения являются основным типом контента, развернутого службой развертывания. Вы можете найти в каталоге, чтобы найти конкретные обновления, доступные для развертывания.

Возможно, вы уже знакомы с каталогом [обновлений Майкрософт,](https://www.catalog.update.microsoft.com/) в котором перечислены обновления программного обеспечения для Windows. Служба развертывания предоставляет собственный каталог [и](/graph/api/resources/windowsupdates-catalog)собирает эквивалентные обновления в едином [каталогеEntry,](/graph/api/resources/windowsupdates-catalogentry) чтобы упростить процессы принятия решений и утверждения.

## <a name="windows-update-categories"></a>Категории обновлений Windows

В общем, существует две категории обновлений высокого Windows 10: обновления функций и обновления качества. В категорию обновлений качества входят обновления безопасности, обновления драйверов и другие.

Каталог служб [развертывания также](/graph/api/resources/windowsupdates-catalog?view=graph-rest-beta&preserve-view=true) классифицировать обновления как обновления функций и качества. [Записи каталога обновлений](/graph/api/resources/windowsupdates-qualityupdatecatalogentry?view=graph-rest-beta&preserve-view=true) качества определяют обновления безопасности и небезопасности определенным образом и исключают обновления драйверов. Обратите внимание, что определение отличается от каталога обновлений Майкрософт. Дополнительные [сведения см. в](#quality-updates) дополнительных сведениях.

Фактически служба развертывания в настоящее время развертывает только обновления функций и обновления качества безопасности, как это определено в каталоге.  В настоящее время служба не развертывает обновления качества без обеспечения безопасности или обновления драйвера.

Дополнительные дополнительные Windows 10 обновления и обслуживание см. в кратком руководстве [по Windows как услуга.](/windows/deployment/update/waas-quick-start)

## <a name="identifying-updates-for-deployment"></a>Определение обновлений для развертывания

Обновления в каталоге обновлений Майкрософт очень детализированный и специфичествен для отдельных продуктов, выпусков и архитектур ЦП. 

Например, следующие два обновления качества безопасности считаются разными выпусками в каталоге обновлений Майкрософт, даже если они отличаются только архитектурой.

| Title                                                                                   | Продукты                           | Классификация   |
|-----------------------------------------------------------------------------------------|------------------------------------|------------------|
| Накопительное обновление 2021-03 для Windows 10 версии 20H2 для **x86-based** Systems (KB5000802) | Windows 10, версия 1903 и более поздние версии | Обновления безопасности |
| Накопительное обновление 2021-03 для Windows 10 версии 20H2 для **x64-based** Systems (KB5000802) | Windows 10, версия 1903 и более поздние версии | Обновления безопасности |

В каталоге, предоставленном службой развертывания Windows для бизнеса, эти обновления объединяются в одну запись.

| Отображаемое имя                                           | Тип                                                     |
|--------------------------------------------------------|----------------------------------------------------------|
| 03.09.2021 - 2021.03 B Обновления безопасности для Windows 10 | microsoft.graph.windowsUpdates.qualityUpdateCatalogEntry |

Это агрегирование упрощает процесс утверждения обновлений в различных установленных базах. Аналогичным образом служба развертывания развертывает выпуски обновлений функций.

### <a name="common-properties"></a>Общие свойства

Все обновления в каталоге службы развертывания имеют следующие общие свойства.

| Свойство                | Описание                                                        |
|-------------------------|--------------------------------------------------------------------|
| id                      | Уникальный идентификатор для записи каталога.                           |
| displayName             | Название обновления программного обеспечения.                                      |
| releaseDateTime         | Дата и время выпуска или обновления обновления.                |
| deployableUntilDateTime | Дата и время, пока обновление не может быть развернуто, если известно.|

### <a name="feature-updates"></a>Обновления функций

Обновления функций в каталоге служб развертывания идентифицированы по версии. Записи совокупные различия между архитектурой (например, x86 против x64) и продуктом (в каталоге обновления Майкрософт все обновления функций для Windows 10 *продукта).*

| Свойство | Описание                                       |
|----------|---------------------------------------------------|
| version  | Версия обновления функций для Windows 10 выпуска.|

Ниже приведены некоторые примеры обновлений функций в каталоге службы развертывания.

| Отображаемое имя                               | Версия |
|--------------------------------------------|---------|
| Обновление функций Windows 10 версии 20H1 | 20H1    |
| Обновление функций Windows 10 версии 1909 г. | 1909    |
| Обновление функций Windows 10 версии 1903 | 1903    |
| Обновление функций для Windows 10, версия 1809 | 1809    |

Как только вы определите нужную версию, назначьте ее в качестве контента развертыванию с помощью [featureUpdateReference](/graph/api/resources/windowsupdates-featureupdatereference) и укажите **свойство версии.**

### <a name="quality-updates"></a>Обновления качества

Обновления качества в каталоге служб развертывания идентифицированы по дате выпуска/времени и классификации обновлений. Записи совокупные различия в архитектуре, продукте *(например, Windows 10, версии 1903* и более поздней версии *Windows 10* *LTSB Windows 10 LTSB)* и соответствующей версии обновления функций.

| Свойство | Описание |
|----------|-------------|
| classification | Классификация (безопасность или небезопасная) обновления качества. |
| releaseDateTime | Дата и время выпуска или обновления обновления. |

В следующей таблице показано сопоставление классификации между каталогом служб развертывания и каталогом обновлений Майкрософт.

| Каталог служб развертывания | Каталог обновлений Майкрософт                                                                                                               |
|------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Безопасность         | Обновление безопасности<br>Критическое обновление<br>Обновление (при необходимости в зависимости)<br>Обслуживание обновления стека (при необходимости в зависимости) |
| Небезопасная     | Update<br>Обслуживание обновления стека                                                                                               |

Записи из каталога обновлений Майкрософт, соответствующие обновлению качества в каталоге службы развертывания и могут `classification = security` `releaseDateTime = 2021-03-09` включать следующие.

| Title                                                                                   | Продукты                           | Классификация   |
|-----------------------------------------------------------------------------------------|------------------------------------|------------------|
| Накопительное обновление 2021-03 для Windows 10 версии 20H2 для x86-based Systems (KB5000802) | Windows 10, версия 1903 и более поздние версии | Обновления безопасности |
| Накопительное обновление 2021-03 для Windows 10 версии 20H2 для x64-based Systems (KB5000802) | Windows 10, версия 1903 и более поздние версии | Обновления безопасности |
| Накопительное обновление 2021-03 для Windows 10 версии 1909 для x86-based Systems (KB500080808) | Windows 10, версия 1903 и более поздние версии | Обновления безопасности |
| Накопительное обновление 2021-03 для Windows 10 версии 1809 для x64-based Systems (KB5000822) | Windows 10 Windows 10 LTSB        | Обновления безопасности |

После идентифицированного обновления назначьте его в качестве контента развертыванию с помощью [qualityUpdateReference](/graph/api/resources/windowsupdates-qualityupdatereference) и укажите свойства **releaseDateTime** и **classification.**
