# <a name="software-updates-with-the-windows-update-for-business-deployment-service"></a>Обновления программного обеспечения с помощью клиентский компонент Центра обновления Windows развертывания для бизнеса

Обновления программного обеспечения являются основным типом содержимого, развертываемого службой развертывания. В каталоге можно найти определенные обновления, доступные для развертывания.

Возможно, вы уже знакомы с каталогом [Центра обновления Майкрософт](https://www.catalog.update.microsoft.com/), в котором перечислены обновления программного обеспечения для Windows. Служба развертывания предоставляет собственный [каталог и объединяет](/graph/api/resources/windowsupdates-catalog) эквивалентные обновления в одном [catalogEntry](/graph/api/resources/windowsupdates-catalogentry) , чтобы упростить процессы принятия решений и утверждения.

## <a name="windows-update-categories"></a>Категории обновлений Windows

Как правило, существует две категории общих Windows 10 обновлений: обновления компонентов и исправления. К категории исправлений относятся обновления системы безопасности, обновления драйверов и другие.

Каталог служб [развертывания также](/graph/api/resources/windowsupdates-catalog?view=graph-rest-beta&preserve-view=true) классифицирует обновления как обновления компонентов и исправлений. [Записи каталога исправлений](/graph/api/resources/windowsupdates-qualityupdatecatalogentry?view=graph-rest-beta&preserve-view=true) определяют обновления безопасности и обновления, не связанные с безопасностью, определенным образом и исключают обновления драйверов. Обратите внимание, что определение отличается от каталога Центра обновления Майкрософт. [Дополнительные сведения см](#quality-updates). в приведенных ниже исправлениях.

Фактически служба развертывания в настоящее время развертывает только обновления компонентов и исправления  безопасности, как определено в каталоге. В настоящее время служба не развертывает исправления, не внося в безопасность, или обновления драйверов.

Дополнительные сведения о Windows 10 и обслуживании см. в кратком руководстве [по Windows как услуга](/windows/deployment/update/waas-quick-start).

## <a name="identifying-updates-for-deployment"></a>Определение обновлений для развертывания

Обновления в каталоге обновлений Майкрософт очень детализированные и относятся к отдельным продуктам, выпускам и архитектурам ЦП. 

Например, следующие два обновления качества безопасности считаются различными выпусками в каталоге центра обновления Майкрософт, даже если они отличаются только архитектурой.

| Title                                                                                   | Продукты                           | Классификация   |
|-----------------------------------------------------------------------------------------|------------------------------------|------------------|
| Накопительный пакет обновления 2021–03 для Windows 10 версии 20H2 для **систем на базе x86** (KB5000802) | Windows 10, версия 1903 и более поздние версии | Безопасность Актуальные данные |
| Накопительный пакет обновления 2021–03 для Windows 10 версии 20H2 для **систем на базе x64** (KB5000802) | Windows 10, версия 1903 и более поздние версии | Безопасность Актуальные данные |

В каталоге, предоставляемом службой клиентский компонент Центра обновления Windows для бизнеса, эти обновления объединяются в одну запись.

| Отображаемое имя                                           | Тип                                                     |
|--------------------------------------------------------|----------------------------------------------------------|
| 09.03.2021– 2021.03 B security Актуальные данные для Windows 10 | microsoft.graph.windowsUpdates.qualityUpdateCatalogEntry |

Это агрегирование упрощает процесс утверждения обновлений в разных установленных базах. Аналогичным образом служба развертывания выполняет сведение выпусков обновлений компонентов.

### <a name="common-properties"></a>Общие свойства

Все обновления в каталоге служб развертывания имеют следующие общие свойства.

| Свойство                | Описание                                                        |
|-------------------------|--------------------------------------------------------------------|
| id                      | Уникальный идентификатор для записи каталога.                           |
| displayName             | Название обновления программного обеспечения.                                      |
| releaseDateTime         | Дата и время выпуска или обновления обновления.                |
| deployableUntilDateTime | Дата и время, пока обновление не может быть развернуто, если известно.|

### <a name="feature-updates"></a>Обновления функций

Обновления компонентов в каталоге служб развертывания определяются по версии. Записи объединяют различия между архитектурой (например, x86 и x64) и продуктом (в каталоге Центра обновления Майкрософт все обновления компонентов предназначены для *Windows 10 продукта).*

| Свойство | Описание                                       |
|----------|---------------------------------------------------|
| version  | Версия обновления компонентов для Windows 10 версии.|

Ниже приведены некоторые примеры обновлений компонентов в каталоге служб развертывания.

| Отображаемое имя                               | Версия |
|--------------------------------------------|---------|
| Обновление компонентов до Windows 10 версии 20H1 | 20H1    |
| Обновление компонентов до Windows 10 версии 1909 | 1909    |
| Обновление компонентов до Windows 10 версии 1903 | 1903    |
| Обновление компонентов для Windows 10, версия 1809 | 1809    |

После определения нужной версии назначьте ее в качестве содержимого для развертывания с помощью [featureUpdateReference](/graph/api/resources/windowsupdates-featureupdatereference) и укажите **свойство версии** .

### <a name="quality-updates"></a>Обновления качества

Исправления в каталоге служб развертывания определяются по дате и времени выпуска и классификации обновлений. Записи объединяют различия между архитектурой, продуктом (например *, Windows 10, версией 1903* и более поздней и Windows 10 и *Windows 10 LTSB*) и соответствующей версией обновления компонентов. 

| Свойство | Описание |
|----------|-------------|
| classification | Классификация (безопасность или не безопасность) обновления качества. |
| releaseDateTime | Дата и время выпуска или обновления обновления. |

В следующей таблице показано сопоставление классификации между каталогом служб развертывания и каталогом Центра обновления Майкрософт.

| Каталог служб развертывания | Каталог обновлений Майкрософт                                                                                                               |
|------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Безопасность         | Обновление безопасности<br>Критическое обновление<br>Обновление (при необходимости в качестве зависимости)<br>Обновление стека обслуживания (при необходимости в качестве зависимости) |
| Не из-за безопасности     | Update<br>Обновление стека обслуживания                                                                                               |

Записи из каталога Центра обновления `classification = security` `releaseDateTime = 2021-03-09` Майкрософт, соответствующие обновлению качества в каталоге служб развертывания, и могут включать в себя следующее.

| Title                                                                                   | Продукты                           | Классификация   |
|-----------------------------------------------------------------------------------------|------------------------------------|------------------|
| Накопительный пакет обновления 2021–03 для Windows 10 версии 20H2 для систем на базе x86 (KB5000802) | Windows 10, версия 1903 и более поздние версии | Безопасность Актуальные данные |
| Накопительный пакет обновления 2021–03 для Windows 10 версии 20H2 для систем на базе x64 (KB5000802) | Windows 10, версия 1903 и более поздние версии | Безопасность Актуальные данные |
| Накопительный пакет обновления 2021–03 для Windows 10 версии 1909 для систем на базе x86 (KB5000808) | Windows 10, версия 1903 и более поздние версии | Безопасность Актуальные данные |
| Накопительный пакет обновления 2021–03 для Windows 10 версии 1809 для систем на базе x64 (KB5000822) | Windows 10, Windows 10 LTSB        | Безопасность Актуальные данные |

После определения нужного обновления назначьте его в качестве содержимого для развертывания с помощью [qualityUpdateReference](/graph/api/resources/windowsupdates-qualityupdatereference) и укажите **releaseDateTime** и **свойства** классификации.
