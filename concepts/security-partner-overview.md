# <a name="partnering-opportunities-with-the-microsoft-graph-security-api"></a>Возможности партнеров с помощью microsoft Graph API безопасности

В этой статье описываются возможности партнеров, включенные microsoft Graph API безопасности. Он предназначен для того, чтобы помочь менеджерам по продуктам и ролям бизнес-разработки понять пути инвестиций и получить ценные сведения о преимуществах партнеров.

## <a name="background"></a>Общие сведения

Большинство организаций работают с высоким объемом данных безопасности и имеют десятки решений по обеспечению безопасности на предприятии, что делает задачу интеграции различных продуктов и служб сложной и сложной. Эти проблемы препятствуют быстрому перемещению организаций при обнаружении и устранении угроз в мире быстрых и вредоносных атак.

Технологические партнеры могут интегрироваться с платформой Майкрософт с помощью microsoft Graph API безопасности для решения этих проблем клиентов.

## <a name="introduction-to-the-microsoft-graph-security-api"></a>Общие сведения о microsoft Graph API безопасности

Microsoft Graph Security API — это единый API, который предоставляет стандартный интерфейс и единую схему для интеграции оповещений от системы безопасности и аналитических данных об угрозах из нескольких источников, обогащает оповещения и данные контекстной информацией, а также автоматизирует операции системы безопасности.

API безопасности входит в состав Microsoft Graph — единого REST API для интеграции данных и аналитики от Майкрософт, а также партнерских продуктов и служб. Благодаря Microsoft Graph клиенты и партнеры могут быстро создавать решения, которые проходят проверку подлинности один раз и используют один вызов API для доступа к аналитическим сведениям о системе безопасности из различных решений и принятия соответствующих мер. При изучении других сущностей [Microsoft Graph](./overview.md) (Microsoft 365, Azure Active Directory, Intune и т. д.) для связывания бизнес-контекста с аналитическими сведениями о безопасности вы можете получить дополнительную ценность.

Корпорация Майкрософт обеспечивает интеграцию технологических партнеров двумя ключевыми способами.

1. Как потребитель информации из Microsoft Graph вы можете дополнить свои решения сведениями, содержащимся в Microsoft Graph, а также использовать microsoft API Graph для выполнения задач от имени клиента.
2. Вы также можете добавлять оповещения и действия в Microsoft Graph вместе с поставщиками Майкрософт.

## <a name="featured-technology-partner-scenarios"></a>Рекомендуемые сценарии партнеров по технологиям

Ниже приведены три ключевых преимущества, которые можно получить путем интеграции с microsoft Graph API безопасности.

1. Ваши клиенты получают преимущества от повышения эффективности безопасности и операций.
2. Ваши клиенты получают преимущества от расширенной информации, предоставляемой вашими и другими интегрированными партнерскими продуктами.
3. Инвестиции в проектирование для технологических партнеров упрощены, а ценность клиента увеличивается благодаря интеграции с microsoft Graph API безопасности.

### <a name="enhance-threat-protection-with-the-microsoft-graph-security-api"></a>Улучшение защиты от угроз с помощью microsoft Graph API безопасности

*Упрощение интеграции оповещений системы безопасности для обнаружения угроз и реагирования на них.*

- Сопоставьте оповещения и обнаружения от поставщиков безопасности Microsoft Graph с обнаружениями, чтобы улучшить результаты исследования и поддерживать автоматизацию.
- Обнаружение и контекст доступа через Microsoft Graph для улучшения реагирования на угрозы — рассмотрение, исследование, исправление.
- Доступ к аналитике угроз клиентов (хэш, IP-адрес, URL-адрес, домен и т. д.) для блокировки или оповещения о вредоносных действиях.

### <a name="streamline-it-and-security-management"></a>Упрощение управления ИТ-ресурсами и безопасностью

*Повышение видимости и упрощение управления жизненным циклом инцидентов.*

- Статистическая обработка оповещений от нескольких поставщиков для создания инцидентов.
- Доступ к дополнительному контексту для информирования о приоритетах оповещений и ответе.
- Синхронизация состояния оповещений между системами, управляющими оповещениями.
- Получите представление о состоянии безопасности и рекомендации по его улучшению с помощью оценки безопасности.

### <a name="share-threat-intelligence-to-enable-custom-detections"></a>Совместное использование аналитики угроз для включения пользовательских обнаружений

*Используйте аналитику угроз, чтобы использовать пользовательские обнаружения в решениях Майкрософт.*

- Автоматически отправлять индикаторы угроз в решения майкрософт по обеспечению безопасности, чтобы включить действия оповещений, блокировки или разрешения.
- Включите быстрые действия для защиты от новых угроз, таких как блочный файл, URL-адрес, домен, IP-адрес в средствах безопасности и рабочих процессах.
- Предоставленный клиентом ti используется только для клиента, предоставляемого, а не для других клиентов Майкрософт.

## <a name="technical-integrations-overview"></a>Общие сведения о технической интеграции

Возможности партнерского API безопасности Microsoft Graph доступны через два основных пути интеграции, которые можно использовать независимо или вместе.  Мы рассмотрим общие требования и предоставьте некоторые сведения о том, как подумать об инвестициях в эти пути, но подробные технические объяснения слева на документах, на которые ссылаются далее в этом документе.

Поддерживаемые сущности:

- **Оповещения —** это "выводы, влияющие на безопасность", а не необработанные данные журнала или другие некоррелтные сведения. [Подробнее](/graph/api/resources/security-api-overview#alerts).
- **Индикаторы угроз**, также называемые индикаторами компрометации или IoCs, представляют данные об известных угрозах, таких как вредоносные файлы, URL-адреса, домены и IP-адреса. Клиенты могут создавать индикаторы с помощью сбора данных внутренней аналитики угроз или получения индикаторов от сообществ аналитики угроз, лицензированных веб-каналов и других источников. [Подробнее](/graph/api/resources/tiindicator).
- **Действия безопасности** позволяют технологическим партнерам предоставлять функциональные возможности через Graph.  Например, если решение для обеспечения безопасности поддерживает возможность блокировки IP-адресов, вы можете предоставить в графе функцию "Блокировать IP-адрес". Другие продукты API безопасности Graph могут вызывать ваше действие через Graph. [Подробнее](/graph/api/resources/securityaction).
- **Оценка безопасности**... [Подробнее](/graph/api/resources/securescores).

### <a name="integrate-your-application-with-the-microsoft-graph-security-api"></a>Интеграция приложения с microsoft Graph API безопасности

Все интегрированные приложения [должны быть зарегистрированы](./auth-register-app-v2.md) в Microsoft Graph. Поддерживаются как приложения, используемые одним клиентом, так и многие клиенты (мультитенантные).  В любом случае клиент должен предоставить согласие для вашего приложения. При вызове Microsoft Graph каждый запрос из приложения будет содержать идентификатор приложения и клиента, которого вы вызываете от имени. Поддерживаются следующие типы запросов:

- **Получение оповещений** — получение сведений об оповещении с фильтрацией по мере необходимости.  Например, показать все оповещения с высоким приоритетом или "все оповещения с высоким приоритетом" для конкретного пользователя, узла и т. д.
- **Обновление состояния оповещения** — включение управления жизненным циклом оповещений.  Например: установка состояния оповещений "разрешено" из "выполняется" или добавление комментариев к оповещению.
- **Получение оценки безопасности** — оценка безопасности Майкрософт — это значение типа кредитной оценки для конфигураций безопасности продуктов Майкрософт.
- **Подписка** — разрешение уведомлений об изменениях оповещений или запросов.
- **Веб-канал** настраиваемых индикаторов угроз — автоматически отправлять индикаторы угроз в решения безопасности Майкрософт для включения действий оповещений, блокировки или разрешения. Используйте microsoft Graph API безопасности напрямую или используйте интеграцию с ведущими платформами аналитики угроз.
- **Вызов действия безопасности Microsoft Graph** — немедленное действие по защите от угроз с помощью сущности Microsoft Graph Security [SecurityActions](/graph/api/resources/securityaction) .

### <a name="enable-others-to-integrate-with-your-products-through-the-microsoft-graph-security-api"></a>Предоставление другим пользователям возможности интеграции с вашими продуктами с помощью microsoft Graph API безопасности

Поставщики безопасности Microsoft Graph делают свои оповещения системы безопасности доступными другим пользователям через Microsoft Graph.  Продукты Майкрософт, которые создают оповещения системы безопасности, имеют поставщиков, которые предоставляют соответствующие оповещения Microsoft Graph. Кроме того, Microsoft Graph API безопасности внешних поставщиков, что позволяет вам, как партнеру по технологиям Майкрософт, совместно использовать соответствующие оповещения системы безопасности из приложений в Microsoft Graph, чтобы клиенты могли использовать их. Помимо оповещений, Microsoft Graph Security SecurityActions позволяет технологическим партнерам предоставлять функциональные возможности через Microsoft Graph.  Например, если решение для обеспечения безопасности поддерживает возможность блокировки IP-адресов, вы можете предоставить функцию "Блокировать IP-адрес" в Microsoft Graph. Другие продукты Microsoft Graph Security могут вызывать ваше действие через Microsoft Graph.

Поставщик безопасности Microsoft Graph — это, по сути, облачная конечная точка, которая отвечает на запросы от microsoft Graph API безопасности и возвращает соответствующие оповещения системы безопасности или выполняет действия для взаимного клиента. Проверка подлинности между клиентами и службами обеспечивает защиту доступа к оповещениям и действиям клиентов.

Сценарии поставщиков широко разнообразны. Курируемый процесс подключения начинается с определения соответствующих сценариев. После того как сценарии будут согласованы, документация, примеры кода и среды разработки будут доступны для поддержки разработки поставщика безопасности Microsoft Graph.
