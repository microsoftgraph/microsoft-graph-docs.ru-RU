# <a name="get-component-in-the-microsoft-graph-toolkit"></a>Получить компонент в microsoft Graph набор средств

Вы можете использовать `mgt-get` любой запрос GET из Microsoft Graph непосредственно в HTML-коде. Компонент не предоставляет пользовательский интерфейс по умолчанию и требует написания шаблона.

## <a name="example"></a>Пример

В следующем примере показано использование компонента для отображения сообщений электронной почты `mgt-get` пользователя. С помощью редактора кода можно увидеть, как свойства и [атрибуты](#properties-and-attributes) изменяют поведение компонента.

## <a name="properties-and-attributes"></a>Свойства и атрибуты

Для изменения поведения компонента можно использовать несколько атрибутов. Единственным требуемым атрибутом `resource` является .

| Атрибут | Свойство  | Описание |
| --- | --- | --- |
| resource | resource | Ресурс, который необходимо получить из Microsoft Graph (например, `/me` ). |
| scopes | scopes | Необязательный массив строк при использовании свойства или области с запятой при использовании атрибута. Компонент будет использовать эти области (с поддерживаемой службой), чтобы убедиться, что пользователь согласился на правильное разрешение. |
| version | version | Необязательная версия API, используемая при создании запроса GET. Значение по умолчанию: `v1.0`.  |
| max-pages | maxPages | Необязательное количество страниц (для ресурсов, поддерживаюющих разбавку). Значение по умолчанию — 3. Если установить для этого параметра значение 0, будут получаться все страницы.  |
| polling-rate | pollingRate | Необязательный номер едисекунд. Если этот компонент установлен, он опрошит URI запроса на обновление за определенный интервал. При использовании запроса delta опрос всегда будет запрашивать API delta. Шаблон обновляется только при изменениях данных. |
| cache-enabled | cacheEnabled | Необязательный атрибут типа Boolean. Если этот набор за установлен, это означает, что ответ от ресурса будет кэш создан. Переопределяется, `refresh()` если он вызван или `pollingRate` используется. Значение по умолчанию: `false`. |
| cache-invalidation-period | cacheInvalidationPeriod | Необязательный номер едисекунд. Если этот срок установлен в сочетании с ним, это значение будет изменено с задержкой до достижения периода недействительной работы `cacheEnabled` кэша. По умолчанию `0` используется период недействительности по умолчанию. |
| type | type | Необязательный тип ожидаемого ответа. Значение по умолчанию: `json`. Поддерживает или `json` `image` (поддерживается только для конечных точек, заканчивающийся на `/photo/value$` ). |
| Недоступно | response | Ответ Microsoft Graph только для чтения в случае успешного запроса.  |
| Недоступно |error| Ошибка только для чтения из Microsoft Graph, если запрос не был успешной. |

## <a name="methods"></a>Методы
| Метод | Описание |
| --- | --- |
| refresh(force?:boolean) | Вызовите метод для обновления данных. По умолчанию пользовательский интерфейс обновляется только при изменениях данных. `true`Передав его, чтобы принудительно обновить компонент.  |


## <a name="events"></a>События
| Событие | Сведения | Описание |
| --- | --- | --- |
| dataChange | Сведения содержат `response` объекты и `error` объекты. | Происходит при изменении отклика или ошибки. |

## <a name="templates"></a>Шаблоны

Компонент `mgt-get` поддерживает несколько [шаблонов,](../customize-components/templates.md) которые можно использовать для определения оформления и оформления. Чтобы указать шаблон, включив элемент в компонент, закажите одно из `<template>` `data-type` следующих значений.

| Тип данных | Контекст данных | Описание |
| --- | --- | --- |
| default | Ответ от Microsoft Graph. | Шаблон по умолчанию необходим для отображения данных, полученных из Microsoft Graph. |
| значение | Элемент данных из возвращенного `value` массива | Используйте шаблон вместо шаблона, если ожидается, что ответ от графа будет содержать массив элементов, таких как сообщения, файлы `value` `default` или **пользователи.**   Шаблон `value` будет автоматически повторяться для каждого элемента, возвращенного ресурсом. Шаблон также начнет отрисовку элементов сразу после их готовности (в отличие `value` от шаблона по умолчанию).|
| error | Ошибка Microsoft Graph. | Этот шаблон будет использоваться при ошибке при создании запроса. |
| loading | Недоступно | Этот шаблон используется во время запроса. |
