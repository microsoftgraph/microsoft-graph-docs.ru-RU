# <a name="accesspackageresourceattributesource-resource-type"></a>тип ресурса accessPackageResourceAttributeSource

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Абстрактный тип, который предоставляет объекты, определяемые свойствами, которые пользователь настраивает со значениями, которые передаются в итоге системы или утверждения запроса.  Он используется в **свойстве attributeSource** [accessPackageResourceAttribute](accesspackageresourceattribute.md). Этот тип наследуется [accessPackageResourceAttributeQuestion](../resources/accesspackageresourceattributequestion.md).
