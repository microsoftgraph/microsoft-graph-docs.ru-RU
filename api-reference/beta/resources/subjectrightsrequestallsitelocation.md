# <a name="subjectrightsrequestallsitelocation-resource-type"></a>Тип ресурса subjectRightsRequestAllSiteLocation

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет свойства запроса прав субъекта, который определяет все сайты (сайты SharePoint, OneDrive для бизнеса сайты и Microsoft Teams каналы) в качестве расположения поиска. Используйте этот тип, если требуется искать данные на всех сайтах.

Наследуется [от subjectRightsRequestSiteLocation](../resources/subjectrightsrequestsitelocation.md).

## <a name="properties"></a>Свойства
Нет

## <a name="relationships"></a>Связи
Отсутствуют.
