# <a name="subjectrightsrequestallmailboxlocation-resource-type"></a>Тип ресурса subjectRightsRequestAllMailboxLocation

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]
