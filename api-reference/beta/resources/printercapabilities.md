# <a name="printercapabilities-complex-type"></a>Сложный тип printerCapabilities

Пространство имен: microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет возможности, сообщаемые принтером.

## <a name="properties"></a>Свойства
| Свойство     | Тип        | Описание |
|:-------------|:------------|:------------|
|contentTypes|Коллекция строк|Список поддерживаемых типов содержимого (MIME), поддерживаемых принтером. Не гарантируется, что служба универсальной печати поддерживает печать всех этих типов MIME.|
|isColorPrintingSupported|Логическое|Значение true, если принтер поддерживает цветовую печать; Значение false в противном случае. Только для чтения.|
|feedOrientations|Коллекция printerFeedOrientation|Список ориентаций веб-канала, поддерживаемых принтером.|
|isPageRangeSupported|Логическое|Значение true, если принтер поддерживает печать по диапазонам страниц; Значение false в противном случае.|
|Качества|Коллекция printQuality|Качества печати, поддерживаемые принтером.|
|dpis|Коллекция Int32|Список разрешений печати в DPI, поддерживаемых принтером.|
|duplexModes|Коллекция printDuplexMode|Список дуплексных режимов, поддерживаемых принтером. Допустимые значения описаны в следующей таблице.|
|queueBufferSizeInBytes|Int32|Максимальный размер очереди задания печати, который может храниться принтером.|
|copiesPerJob|[integerRange](integerrange.md)|Диапазон копий на задание, поддерживаемый принтером.|
|finishings|Коллекция printFinishing|Завершение процессов, поддерживаемых принтером для печатного документа.|
|mediaColors|Коллекция строк|Цвет мультимедиа (т. е. бумаги), поддерживаемый принтером.|
|mediaTypes|Коллекция строк|Типы носителей, поддерживаемые принтером.|
|mediaSizes|Коллекция строк|Размеры носителей, поддерживаемые принтером. Поддерживает стандартные имена размеров для размеров носителя ISO и ANSI. Допустимые значения приведены в следующей [таблице](#mediasizes-values).|
|pagesPerSheet|Коллекция Int32|Поддерживаемое количество входных страниц для одного просмотра.|
|Ориентации|Коллекция printOrientation|Ориентации печати, поддерживаемые принтером. Допустимые значения описаны в следующей таблице.|
|inputBins|Коллекция объектов string|Поддерживаемые входные ячейки для принтера.|
|outputBins|Коллекция строк|Поддерживаемые принтером выходные ячейки (области).|
|supportsFitPdfToPage|Логическое|Значение true, если принтер поддерживает масштабирование PDF-страниц в соответствии с размером печатных носителей; Значение false в противном случае.|
|multipageLayouts|Коллекция printMultipageLayout|Направления презентации, поддерживаемые принтером. Поддерживаемые значения описаны в следующей таблице.|
|colorModes|Коллекция printColorMode|Цветовые режимы, поддерживаемые принтером. Допустимые значения описаны в следующей таблице.|
|topMargins|Коллекция Int32|Список поддерживаемых верхних полей (в микронах) для принтера.|
|bottomMargins|Коллекция Int32|Список поддерживаемых нижних полей (в микронах) для принтера.|
|rightMargins|Коллекция Int32|Список поддерживаемых правых полей (в микронах) для принтера.|
|leftMargins|Коллекция Int32|Список поддерживаемых левых полей (в микронах) для принтера.|
|разбор по копиям;|Логическое|Значение true, если принтер поддерживает сортировку при печати muliple-копий многошагового документа; Значение false в противном случае.|
|scalings|Коллекция printScaling|Поддерживаемые масштабирование печати.|

## <a name="printerfeedorientation-values"></a>Значения printerFeedOrientation

|Элемент|Значение|Описание|
|:---|:---|
|longEdgeFirst|0|Принтер будет использовать листы из активной области в альбомной ориентации с длинным краем листа.|
|shortEdgeFirst|1|Принтер будет использовать листы из активной области в книжной ориентации с коротким краем листа.|
|unknownFutureValue|2|Значение sentinel для развиваемого перечисления. Не следует использовать.|

### <a name="printquality-values"></a>Значения printQuality

|Элемент|Значение|Описание|
|:---|:---|
|Низкой|0|Принтер выведет задание с низким качеством (обычно называется черновиком).|
|medium|1|Принтер выведет задание с помощью medim (обычного) качества.|
|Высокой|2|Принтер выведет задание, используя высокое (обычно известное как "лучшее" или "отличное") качество.|
|unknownFutureValue|3|Значение sentinel для развиваемого перечисления. Не следует использовать.|

### <a name="printduplexmode-values"></a>Значения printDuplexMode

|Элемент|Значение|Описание|
|:---|:---|:---|
|FlipOnLongEdge|0|Принтер напечатает двунаправленную печать и пролистает документы вдоль длинного края.|
|FlipOnShortEdge|1|Принтер напечатает двунаправленную печать и пролистает документы вдоль короткого края.|
|oneSided|2|Принтер выведет однобокий принтер.|

### <a name="printfinishing-values"></a>значения printFinishing

|Элемент|Значение|Описание|
|:---|:---|:---|
|Нет|3|Нет завершений. Включение этого значения эквивалентно предоставлению пустой коллекции завершений.|
|Основных|4|Прикрепите документ с помощью конфигурации устаревания принтера по умолчанию.|
|Удар|5|С помощью конфигурации принтера по умолчанию замейте документ.|
|Крышка|6 |Примените обложку к документу.|
|Привязки|7 |Привяжете документ с помощью конфигурации привязки принтера по умолчанию.|
|saddleStitch|8 |Запечатав документ с помощью конфигурации совмещаемой конфигурации принтера по умолчанию.|
|совмещаемая|9 |Совмещайте документ по краям, используя конфигурацию совмещаемой работы принтера по умолчанию.|
|stapleTopLeft;|20|Скрепите документ в левом верхнем углу.|
|stapleBottomLeft;|21|Скрепите документ в левом нижнем углу.|
|stapleTopRight|22|Скрепите документ в правом верхнем углу.|
|основные боттомыRight|23|Скрепите документ в правом нижнем углу.|
|совмещатьLeftEdge|24|Совмещайте документ по левому краю.|
|совмещаемая таблица|25|Граничное сшивание документа по верхнему краю.|
|висячее|26|Совмещайте документ по правому краю.|
|stitchBottomEdge|27|Граничное сшивание документа вдоль нижнего края.|
|stapleDualLeft;|28|Дважды скрепите документ вдоль левого края.|
|stapleDualTop;|29|Дважды скрепите документ вдоль верхнего края.|
|stapleDualRight;|30|Дважды скрепите документ вдоль правого края.|
|stapleDualBottom;|31|Дважды скрепите документ вдоль нижнего края.|
|unknownFutureValue|32|Значение sentinel для развиваемого перечисления. Не следует использовать.|

## <a name="printorientation-values"></a>Значения printOrientation

|Элемент|Значение|Описание|
|:---|:---|:---|
|Портрет|3|Принтер будет печатать показы в "книжной" ориентации.|
|Пейзаж|4|Принтер будет печатать показы в альбомной ориентации.|
|reverseLandscape|5|Принтер будет печатать показы в ориентации "обратная альбомная ориентация".|
|reversePortrait|6 |Принтер будет печатать показы в ориентации "обратная книжная".|

### <a name="printmultipagelayout-values"></a>Значения printMultipageLayout

|Элемент|Значение|Описание|
|:---|:---|:---|
|clockwiseFromTopLeft|0|Расположите страницы в сетке по часовой стрелке, начиная с левого верхнего края.|
|counterclockwiseFromTopLeft|1|Расположите страницы в сетке против часовой стрелки, начиная с левого верхнего края.|
|counterclockwiseFromTopRight|2|Расположите страницы в сетке против часовой стрелки, начиная с правого верхнего.|
|clockwiseFromTopRight|3|Расположите страницы в сетке по часовой стрелке, начиная с правого верхнего.|
|counterclockwiseFromBottomLeft|4|Расположите страницы в сетке против часовой стрелки, начиная с нижнего левого края.|
|clockwiseFromBottomLeft|5|Расположите страницы в сетке по часовой стрелке, начиная с нижнего левого края.|
|counterclockwiseFromBottomRight|6 |Расположите страницы в сетке против часовой стрелки, начиная с нижнего правого.|
|clockwiseFromBottomRight|7 |Расположите страницы в сетке по часовой стрелке, начиная с правого нижнего.|

### <a name="printcolormode-values"></a>Значения printColorMode

|Элемент|Значение|Описание|
|:---|:---|:---|
|blackAndWhite|0|Черно-белый (используйте только черный материал маркера).)|
|Оттенках серого|1|Оттенки серого (могут использовать какой-либо материал цветового маркера).)|
|color|2|Цвет (используйте любое сочетание материалов маркера для создания цветового впечатления).|
|Авто|3|Разрешите принтеру выбрать цветовой режим для использования.|

### <a name="printscaling-values"></a>значения printScaling

|Элемент|Значение|Описание|
|:---|:---|:---|
|Авто|0|Если документ больше запрашиваемого носителя и поля не равен нулю, принтер масштабирует документ так же, **как и размер** printScaling. В противном случае принтер масштабирует документ с помощью **параметра printScaling** заливки. Если размер документа меньше запрашиваемого носителя, используется параметр printScaling none.|
|shrinkToFit|1|Если размер документа превышает размер запрашиваемого носителя, принтер масштабирует документ так же, как **и размер** printScaling. В противном случае принтер масштабирует документ так же, как **и тип printScaling** .|
|fill|2|Принтер масштабирует документ, чтобы заполнить нужный размер носителя, сохраняя пропорции, но потенциально обрезая части документа.|
|Подходят|3|Принтер масштабирует документ в соответствии с областью печати требуемого размера носителя, сохраняя пропорции данных документа без обрезки документа.|
|Нет|4|Принтер не масштабируется в соответствии с размером запрашиваемого носителя. Если размер документа превышает запрашиваемый носитель, принтер центрирует и обрезает полученные выходные данные. Если документ меньше, чем запрашиваемый носитель, принтер размещает полученные выходные данные.|
|unknownFutureValue|5|Значение sentinel для развиваемого перечисления. Не следует использовать.|
