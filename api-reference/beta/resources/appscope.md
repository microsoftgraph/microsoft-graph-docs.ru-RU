# <a name="appscope-resource-type"></a>Тип ресурса appScope

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Область назначения роли определяет набор ресурсов, к которым был предоставлен доступ для основного. Область приложения — это область, определенная и понятная определенным приложением. Другим типом области является область каталога. Области каталогов — это общие области, хранимые в каталоге, которые понимаются несколькими приложениями. 

Он используется как для одного субъекта, одной сущности области, так и для нескольких субъектов с несколькими сущностями области.

## <a name="methods"></a>Методы
Нет

## <a name="properties"></a>Свойства

| Свойство | Тип | Описание |
|:-------- |:---- |:----------- |
| id | string | ИД контейнера или ресурса для конкретного приложения, представляющего область назначения. Обычно не изменяемый ид ресурса. Область назначения определяет набор ресурсов, к которым был предоставлен доступ для основного. Это свойство является обязательной. |
| type | Строка | Описывает тип ресурса для конкретного приложения, представленного областью приложения. Предоставляется для отображения, чтобы пользовательский интерфейс передавал пользователю тип ресурса приложения, представленного областью приложения. Это свойство только для чтения. |
| displayName | string | Отображает имя ресурса для конкретного приложения, представленного областью приложения. Предоставляется для отображения, так как appScopeId часто является неусвояемым, нечитаемым для чтения ид. Это свойство только для чтения. |

## <a name="relationships"></a>Связи
