# <a name="use-the-outlook-mail-rest-api"></a><span data-ttu-id="554ba-103">Использование REST API почты Outlook</span><span class="sxs-lookup"><span data-stu-id="554ba-103">Use the Outlook mail REST API</span></span>

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

<span data-ttu-id="554ba-p101">Microsoft Graph позволяет вашему приложению получать авторизованный доступ к данным почты Outlook в личной или корпоративной учетной записи. For more information on access tokens, app registration, and delegated and application permissions, see [Authentication and authorization basics](/graph/auth/auth-concepts).</span></span>
