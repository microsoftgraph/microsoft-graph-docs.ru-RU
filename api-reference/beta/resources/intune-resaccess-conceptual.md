# <a name="manage-resource-access-certificates-in-intune"></a>Управление сертификатами доступа к ресурсам в Intune

Пространство имен: microsoft.graph

> **Важно!** API бета-версии (/beta) в Microsoft Graph проходят тестирование и могут быть изменены. Использование этих API в производственных приложениях не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Вы можете импортировать сертификаты PFX с помощью Microsoft API Graph для Intune.

Для управления регистрацией в Intune используются перечисленные ниже ресурсы Graph.

- [Сертификат PFX пользователя](intune-raimportcerts-userpfxcertificate.md)
- [Назначение пользовательского PFX](intune-raimportcerts-userpfxintendedpurpose.md)
- [Схема заполнения PFX пользователя](intune-raimportcerts-userpfxpaddingscheme.md)
