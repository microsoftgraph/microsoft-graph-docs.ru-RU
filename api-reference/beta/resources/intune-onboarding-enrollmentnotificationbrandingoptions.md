# <a name="enrollmentnotificationbrandingoptions-enum-type"></a>Тип перечисления enrollmentNotificationBrandingOptions

Пространство имен: microsoft.graph

> **Важно:** API Graph Майкрософт в версии /beta могут быть изменены; использование в рабочей области не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Параметры фирменной символики для шаблона сообщения. Фирменная символика определяется в консоли администрирования Intune.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|Нет|0|Без фирменной символики.|
|includeCompanyLogo|1|Включите логотип компании.|
|includeCompanyName|2|Включите название компании.|
|includeContactInformation|4|Включите контактные данные.|
|includeCompanyPortalLink|8 |Включить Корпоративный портал ссылку.|
|includeDeviceDetails|16|Включите сведения об устройстве.|
