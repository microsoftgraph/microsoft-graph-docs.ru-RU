# <a name="macosminimumoperatingsystem-resource-type"></a>тип ресурса macOSMinimumOperatingSystem

Пространство имен: microsoft.graph

> **Важно:** Microsoft Graph API в /бета-версии могут изменяться; использование продукции не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Минимальная операционная система, необходимая для приложения MacOS.

## <a name="properties"></a>Свойства
|Свойство|Тип|Описание|
|:---|:---|:---|
|v10_7|Boolean|Mac OS 10.7 или более поздней.|
|v10_8|Boolean|Mac OS 10.8 или более поздней.|
|v10_9|Boolean|Mac OS 10.9 или более поздней.|
|v10_10|Boolean|Mac OS 10.10 или более поздней.|
|v10_11|Boolean|Mac OS 10.11 или более поздней.|
|v10_12|Boolean|Mac OS 10.12 или более поздней.|
|v10_13|Boolean|Mac OS 10.13 или более поздней.|
|v10_14|Boolean|Mac OS 10.14 или более поздней.|
|v10_15|Boolean|Mac OS 10.15 или более поздней.|
|v11_0|Boolean|Mac OS 11.0 или более поздней.|
|v12_0|Boolean|Mac OS 12.0 или более поздней.|

## <a name="relationships"></a>Связи
Нет
