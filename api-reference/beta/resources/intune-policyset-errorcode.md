# <a name="errorcode-enum-type"></a>Тип кода ошибки

Пространство имен: microsoft.graph

> **Важно:** Microsoft Graph API в /бета-версии могут изменяться; использование продукции не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Пока не задокументировано.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|noError|0|Значение по умолчанию, чтобы указать отсутствие ошибки.|
|несанкционированные|1|Текущий пользователь не имеет доступа из-за отсутствия разрешений RBAC на ресурсе.|
|notFound|2|Текущий пользователь не имеет доступа из-за отсутствия тегов области RBAC на ресурсе.|
|deleted|3|Ресурс удален.|
