# <a name="devicescopeoperator-enum-type"></a>Тип перечисления deviceScopeOperator

Пространство имен: microsoft.graph

> **Важно:** API Graph Майкрософт в версии /beta могут быть изменены; использование в рабочей области не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Оператор запроса конфигурации области устройства. Возможные значения: equals, notEquals, contains, notContains, greaterThan, lessThan. Значение по умолчанию: равно.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|Нет|0|Для конфигурации области устройства оператор не задается.|
|Равно|1|Оператор для используемого запроса конфигурации устройства (Equals).|
|unknownFutureValue|2|Значение заполнителя для будущих перечислений расширения, таких как notEquals, contains, notContains, greaterThan, lessThan.|
