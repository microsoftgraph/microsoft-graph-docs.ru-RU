# <a name="windowsmanagedappclipboardsharinglevel-enum-type"></a>Тип перечисления windowsManagedAppClipboardSharingLevel

Пространство имен: microsoft.graph

> **Важно:** API Graph Майкрософт в версии /beta могут быть изменены; использование в рабочей области не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Представляет уровень, на котором буфер обмена устройства может совместно использоваться приложениями.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|anyDestinationAnySource|0|Пользователи организации могут вставлять и вырезать или копировать данные в любую учетную запись, документ, расположение или приложение.|
|Нет|1|Пользователи организации не могут вырезать, копировать или вставлять данные во внешние учетные записи, документы, расположения или приложения из контекста организации или из него.|
