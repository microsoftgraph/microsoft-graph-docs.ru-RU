> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Поддержка двоичных операторов для выражений условий управления.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|или|0|Оценивает набор операндов как истинный, если и только если один или несколько его операндов верны.|
|и|1|Оценивает набор operands как истинный, если и только если все его операнды верны.|
