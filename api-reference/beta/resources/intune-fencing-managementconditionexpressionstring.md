# <a name="managementconditionexpressionstring-resource-type"></a>тип ресурса managementConditionExpressionString

Пространство имен: microsoft.graph

> **Важно:** Microsoft Graph API в /бета-версии могут изменяться; использование продукции не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Строка выражения условий управления — это строковое представление выражения условий управления.


Наследует от [managementConditionExpression](../resources/intune-fencing-managementconditionexpression.md)

## <a name="properties"></a>Свойства
|Свойство|Тип|Описание|
|:---|:---|:---|
|значение|String|Строка строки выражения выражений условия управления.|

## <a name="relationships"></a>Связи
Нет
