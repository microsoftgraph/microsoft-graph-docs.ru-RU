# <a name="mobileapprelationshiptype-enum-type"></a>тип enum mobileAppRelationshipType

Пространство имен: microsoft.graph

> **Важно:** Microsoft Graph API в /бета-версии могут изменяться; использование продукции не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Указывает, является ли целью отношений родитель или ребенок в отношениях.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|child|0|Указывает, что целью отношений является ребенок в отношениях.|
|родитель|1|Указывает, что целью отношений является родитель в отношениях.|
