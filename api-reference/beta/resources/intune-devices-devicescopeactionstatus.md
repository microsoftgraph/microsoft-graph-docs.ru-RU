# <a name="devicescopeactionstatus-enum-type"></a>Тип перечисления deviceScopeActionStatus

Пространство имен: microsoft.graph

> **Важно:** API Graph Майкрософт в версии /beta могут быть изменены; использование в рабочей области не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Указывает состояние предпринятого действия области устройства.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|Сбой при|0|Указывает, что не удалось активировать действие области устройства.|
|Удалось|1|Указывает, что действие области устройства успешно активируется.|
|unknownFutureValue|2|Значение заполнителя для будущего расширения.|
