# <a name="use-the-employee-learning-api-to-integrate-with-viva-learning"></a>Использование API обучения сотрудников для интеграции с Viva Обучение

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Viva Обучение — это центр обучения сотрудников в Microsoft Teams, который позволяет сотрудникам интегрировать обучение и создание навыков в рабочий день. В Viva Обучение команды могут обнаруживать, делиться, рекомендовать и учиться на библиотеках содержимого, предоставляемых как их организацией, так и партнерами. Они могут быстро получать доступ к назначенному или недавно завершенному содержимому обучения и выполнять все это, не выходя из Microsoft Teams.

API обучения сотрудников в Microsoft Graph позволяет приложениям делать содержимое из системы управления обучением (LMS) или поставщика обучения доступным в Viva Обучение. Используйте следующие ресурсы для интеграции с Viva Обучение:

- [Поставщик обучения](learningprovider.md) для управления поставщиками обучения, включая регистрацию, включение, отключение или удаление поставщиков.
- [Обучающее](learningcontent.md) содержимое для отправки метаданных обучающего содержимого и управления ими из LMS или поставщика обучения в Viva Обучение.


## <a name="how-do-integrations-work"></a>Как работают интеграции?
Администраторы могут интегрировать содержимое и записи учащихся (задания сотрудников и завершенные записи курса) из LMS или поставщика обучения с Viva Обучение, используя встроенные интеграции, где Viva Обучение может извлекать метаданные содержимого и учебные записи из LMS или поставщика обучения. Вы также можете [использовать API](#use-cases-for-the-employee-learning-api-in-microsoft-graph) обучения сотрудников в Microsoft Graph для отправки метаданных содержимого из LMS или поставщика обучения в Viva Обучение. 

Когда метаданные содержимого, назначения сотрудников и их записи курса отправляются в Viva Обучение, они отображаются Viva Обучение в следующих расположениях:
- Содержимое отображается на домашней странице в разделе **"Обзор курсов** **— поставщики** ".
- Задания и записи завершенных курсов отображаются на  странице "Мое обучение" в разделах **"** Назначено" и **"Завершено**" соответственно.

## <a name="authorization"></a>Авторизация
С помощью соответствующих делегированных разрешений на обучение сотрудников приложения ваше приложение может использовать API обучения сотрудников для управления поставщиками обучения и их содержимым для центра обучения в клиенте.[](/graph/permissions-reference#employee-learning-permissions) Дополнительные сведения о маркерах доступа, регистрации приложений, делегированных разрешениях и разрешениях приложений см. в разделе "Основные сведения о проверке подлинности [и авторизации"](/graph/auth/auth-concepts).

## <a name="use-cases-for-the-employee-learning-api-in-microsoft-graph"></a>Варианты использования API обучения сотрудников в Microsoft Graph
Используйте API обучения сотрудников для выполнения следующих действий:
- Зарегистрируйте поставщика в [Viva Обучение](../api/employeeexperience-post-learningproviders.md), укажите отображаемое имя, квадратный логотип, отображаемый на карточке содержимого обучения, и длинный логотип, отображаемый на странице сведений,  который требуется для отображения содержимого поставщика в Viva Обучение. Возвращенный идентификатор регистрации можно использовать для выполнения последующих вызовов приема содержимого.
- Включите или отключите регистрацию и обновите отображаемое имя и URL-адреса логотипа для поставщика.  
- [Получите сведения о поставщике в](../api/learningprovider-get.md) Viva Обучение для определенного **идентификатора регистрации**.  
- [Получение списка регистраций поставщиков](../api/employeeexperience-list-learningproviders.md) в Viva Обучение для клиента.
- [Удаление регистрации](../api/employeeexperience-delete-learningproviders.md) поставщика в Viva Обучение.
- [Отправьте метаданные содержимого](../api/learningcontent-update.md) в Viva Обучение, чтобы содержимое было доступно в Viva Обучение для использования пользователями.  
- [Получение списка метаданных](../api/learningprovider-list-learningcontents.md) принимаемых содержимого поставщика с помощью **идентификатора регистрации** поставщика.  
- [Получение указанных метаданных](../api/learningcontent-get.md) приема содержимого поставщика в Viva Обучение.
- [Удалите указанные метаданные](../api/learningprovider-delete-learningcontents.md) принимаемых содержимого поставщика.

>**Примечание**. Задания и записи завершенных курсов пока недоступны в API обучения сотрудников.
