# <a name="emailsyncduration-enum-type"></a>тип enum emailSyncDuration

Пространство имен: microsoft.graph

> **Важно:** Microsoft Graph API в /бета-версии могут изменяться; использование продукции не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Возможные значения для синхронизации электронной почты.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|userDefined|0|Значение User Defined, значение по умолчанию, без намерения.|
|oneDay|1|Синхронизация одного дня электронной почты.|
|threeDays|2|Синхронизация трех дней электронной почты.|
|oneWeek|3|Синхронизация одной недели электронной почты.|
|twoWeeks|4 |Синхронизация двух недель электронной почты.|
|oneMonth|5 |Синхронизация одного месяца электронной почты.|
|неограниченно|6 |Синхронизация неограниченной продолжительности электронной почты.|
