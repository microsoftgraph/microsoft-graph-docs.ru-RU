# <a name="managementconditionexpression-resource-type"></a>тип ресурса managementConditionExpression

Пространство имен: microsoft.graph

> **Важно:** Microsoft Graph API в /бета-версии могут изменяться; использование продукции не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Выражение условия управления — это выражение, которое при оценке создает значение boolean, то есть истинное или ложное, указывающее на активацию или отключение заявления об условиях управления. Выражение условий управления может состоять из сочетания переменных выражений и операторов выражений со значением boolean.
