# <a name="androiddeviceownerrequiredpasswordunlock-enum-type"></a>Тип перечисления androidDeviceOwnerRequiredPasswordUnlock

Пространство имен: microsoft.graph

> **Важно:** API Graph Майкрософт в версии /beta могут быть изменены; использование в рабочей области не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Перечисление, представляющее возможные значения для обязательной разблокировки пароля.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|deviceDefault|0|Время ожидания до обязательной строгой проверки подлинности устанавливается по умолчанию на устройстве.|
|Ежедневно|1|Время ожидания до обязательной строгой проверки подлинности равно 24 часам.|
|unkownFutureValue|2|Неизвестное будущее значение (зарезервировано, не используется сейчас)|
