# <a name="devicemanagementconfigurationsettingdependedonby-resource-type"></a>Тип ресурса deviceManagementConfigurationSettingDependedOnBy

Пространство имен: microsoft.graph

> **Важно:** API Microsoft Graph в версии /beta могут быть изменены; использование в рабочей области не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Н/Д

## <a name="properties"></a>Свойства
|Свойство|Тип|Описание|
|:---|:---|:---|
|dependedOnBy|String|Идентификатор дочернего параметра, который зависит от текущего параметра|
|обязательно|Логическое|Значение, определяющее, требуется ли дочерний параметр на основе выбора родительского параметра.|
