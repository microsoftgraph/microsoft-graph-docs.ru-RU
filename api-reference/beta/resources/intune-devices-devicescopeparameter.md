# <a name="devicescopeparameter-enum-type"></a>Тип перечисления deviceScopeParameter

Пространство имен: microsoft.graph

> **Важно:** API Graph Майкрософт в версии /beta могут быть изменены; использование в рабочей области не поддерживается.

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Параметр конфигурации области устройства. В будущем для добавления дополнительного параметра будет истек срок действия. Например, параметр области устройства может быть версией ОС, типом диска, производителем устройства, моделью устройства или тегом области. Значение по умолчанию: scopeTag.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|Нет|0|Параметр области устройства не задается|
|scopeTag|1|используйте тег области в качестве параметра для конфигурации области устройства.|
|unknownFutureValue|2|Значение заполнителя для будущего расширения.|
