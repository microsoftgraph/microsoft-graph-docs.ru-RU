# <a name="sharepoint-resource-type"></a>Тип ресурса sharepoint

Пространство имен: microsoft.graph.tenantAdmin

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Представляет контейнер для административных ресурсов для управления параметрами на уровне клиента для SharePoint и OneDrive.

## <a name="properties"></a>Свойства
Нет
