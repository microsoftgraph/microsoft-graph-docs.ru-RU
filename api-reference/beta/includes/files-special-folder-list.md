Следующие специальные имена папок доступны в OneDrive и OneDrive для бизнеса.

| Имя        | Идентификатор папки    | Описание                                                              |
|:------------|:-------------|:-------------------------------------------------------------------------|
| Документы   | `documents`  | Папка "Документы".                                                    |
| Фотографии      | `photos`     | Папка "Фотографии".                                                       |
| Альбом камеры | `cameraroll` | Папка для резервных копий альбома камеры.                                           |
| Корневая папка приложения    | `approot`    | Личная папка приложения. Обычно в `/Apps/{Application Name}` . |
| Музыка       | `music`      | Папка "Музыка".                                                        |
| Записи  | `recordings` | Папка Записи. Этот параметр доступен только в OneDrive для бизнеса и SharePoint Online. |
