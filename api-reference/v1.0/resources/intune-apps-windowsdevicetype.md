# <a name="windowsdevicetype-enum-type"></a>Тип перечисления windowsDeviceType

Пространство имен: microsoft.graph

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Содержит свойства для типа устройства Windows.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|none|0|Флаги не задано.|
|Рабочий стол|1|Поддерживается ли тип устройства Windows для настольных компьютеров.|
|мобильный|2|Поддерживается ли тип мобильного устройства Windows.|
|Голографической|4|Поддерживается ли тип устройства Holographic Windows.|
|team|8 |Поддерживается ли тип устройства Team Windows.|
