# <a name="use-the-outlook-mail-rest-api"></a><span data-ttu-id="7da0b-103">Использование REST API почты Outlook</span><span class="sxs-lookup"><span data-stu-id="7da0b-103">Use the Outlook mail REST API</span></span>

<span data-ttu-id="7da0b-p101">Microsoft Graph позволяет вашему приложению получать авторизованный доступ к данным почты Outlook в личной или корпоративной учетной записи. Имея соответствующие делегированные разрешения или [разрешения почты приложения](/graph/permissions-reference#mail-permissions), приложение может получать доступ к данным почты вошедшего пользователя или любого пользователя в клиенте. For more information on access tokens, app registration, and delegated and application permissions, see [Authentication and authorization basics](/graph/auth/auth-concepts).</span></span>
