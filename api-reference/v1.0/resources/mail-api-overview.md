# <a name="use-the-outlook-mail-rest-api"></a><span data-ttu-id="b4388-103">Использование REST API почты Outlook</span><span class="sxs-lookup"><span data-stu-id="b4388-103">Use the Outlook mail REST API</span></span>

<span data-ttu-id="b4388-p101">Microsoft Graph позволяет вашему приложению получать авторизованный доступ к данным почты Outlook в личной или корпоративной учетной записи. Имея соответствующие делегированные разрешения или [разрешения почты приложения](/graph/permissions-reference#mail-permissions), приложение может получать доступ к данным почты вошедшего пользователя или любого пользователя в клиенте. Дополнительные сведения о маркерах доступа, регистрации приложения, а также делегированных разрешениях и разрешениях приложения см. в статье [Основные сведения о проверке подлинности и авторизации](/graph/auth/auth-concepts).</span><span class="sxs-lookup"><span data-stu-id="b4388-p101">Microsoft Graph lets your app get authorized access to a user's Outlook mail data in a personal or organization account. With the appropriate delegated or application [mail permissions](/graph/permissions-reference#mail-permissions), your app can access the mail data of the signed-in user or any user in a tenant. For more information on access tokens, app registration, and delegated and application permissions, see [Authentication and authorization basics](/graph/auth/auth-concepts).</span></span>
