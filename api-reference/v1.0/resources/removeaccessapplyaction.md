# <a name="removeaccessapplyaction-resource-type"></a>Тип ресурса removeAccessApplyAction

Пространство имен: microsoft.graph

Удаляет доступ к ресурсу для тех, кто отказано в проверке доступа для этого ресурса. Это параметр по умолчанию, выбранный для [accessReviewApplyAction](../resources/accessreviewapplyaction.md) , если параметр не передается.

Наследуется [от accessReviewApplyAction](../resources/accessreviewapplyaction.md).

## <a name="properties"></a>Свойства
Нет

## <a name="relationships"></a>Связи
Отсутствуют.
