# <a name="targetapplicationowners-resource-type"></a>тип ресурса targetApplicationOwners

Пространство имен: microsoft.graph

Этот тип, используемый в политике назначения пакетов доступа, наследуется из [subjectSet](../resources/subjectset.md) и указывает, что владельцы приложений могут запрашивать пакет доступа от имени этого приложения.


## <a name="properties"></a>Свойства

Нет
## <a name="relationships"></a>Связи
Отсутствуют.
