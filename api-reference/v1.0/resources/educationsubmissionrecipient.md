# <a name="educationsubmissionrecipient-resource-type"></a>тип ресурса educationSubmissionRecipient

Пространство имен: microsoft.graph

Абстрактный класс, который представляет различные наборы пользователей, которым может быть назначена отправка. 

Возможные значения включают отдельных лиц или групп. Это используется в **свойстве submission.recipient.**
