# <a name="prereleasefeatures-enum-type"></a>Тип перечисления prereleaseFeatures

Пространство имен: microsoft.graph

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Возможные значения для функций предварительной версии.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|userDefined|0|Определяемое пользователем значение по умолчанию, без намерения.|
|settingsOnly|1|Параметры только функции предварительного выпуска.|
|settingsAndExperimentations|2|Функции предварительной версии параметров и экспериментов.|
|notAllowed|3|Функции предварительной версии не разрешены.|
