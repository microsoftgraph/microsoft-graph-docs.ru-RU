## <a name="use-extended-properties-or-open-extensions"></a><span data-ttu-id="d5087-117">Что использовать — расширенные свойства или открытые расширения</span><span class="sxs-lookup"><span data-stu-id="d5087-117">Use extended properties or open extensions?</span></span>
