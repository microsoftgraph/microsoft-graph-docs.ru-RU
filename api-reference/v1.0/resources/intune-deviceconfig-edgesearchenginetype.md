# <a name="edgesearchenginetype-enum-type"></a>Тип перечисления edgeSearchEngineType

Пространство имен: microsoft.graph

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Позволяет ИТ-администраторам задать предопределенную поисковую систему по умолчанию для MDM-Controlled устройств

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|default|0|Использует параметры фабрики Edge для назначения поисковой системы по умолчанию в соответствии с пользовательским рынком|
|Bing|1|Задает Bing в качестве поисковой системы по умолчанию|
