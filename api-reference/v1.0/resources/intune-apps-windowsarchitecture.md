# <a name="windowsarchitecture-enum-type"></a>Тип перечисления windowsArchitecture

Пространство имен: microsoft.graph

> **Примечание.** API Microsoft Graph для Intune требует наличия [активной лицензии Intune](https://go.microsoft.com/fwlink/?linkid=839381) для клиента.

Содержит свойства для архитектуры Windows.

## <a name="members"></a>Элементы
|Элемент|Значение|Описание|
|:---|:---|:---|
|none|0|Флаги не задано.|
|x86|1|Поддерживается ли тип архитектуры Windows X86.|
|x64|2|Поддерживается ли тип архитектуры Windows X64.|
|Руку|4|Поддерживается ли тип архитектуры Arm Windows.|
|Нейтральных|8 |Поддерживается ли нейтральный тип архитектуры Windows.|
